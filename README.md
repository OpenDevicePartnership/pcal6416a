# PCAL6416A Rust Device Driver

A `#[no_std]` platform-agnostic driver for the [PCAL6416A](https://www.nxp.com/docs/en/data-sheet/PCAL6416A.pdf) IO Expander

## MSRV

Currently, rust `1.81` and up is supported.
